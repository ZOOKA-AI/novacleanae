### الإبلاغ عن ثغرة أمنية

إذا اكتشفت ثغرة أمنية في المشروع:

1. **لا تنشرها علناً**
2. أرسل تقريراً خاصاً إلى: security@zooka-ai.com
3. قدم وصفاً تفصيلياً للثغرة
4. سنرد خلال 48 ساعة

### أفضل الممارسات الأمنية

- لا تشارك مفاتيح API في الكود
- استخدم `.env.local` للمتغيرات الحساسة
- حدّث الحزم بانتظام
- راجع تحذيرات الأمان

### Reporting a Vulnerability

If you discover a security vulnerability:

1. **Do not disclose it publicly**
2. Send a private report to: security@zooka-ai.com
3. Provide detailed description
4. We will respond within 48 hours

### Security Best Practices

- Don't share API keys in code
- Use `.env.local` for sensitive data
- Update packages regularly
- Review security warnings
